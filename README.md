# CharLSTM
Character level generative recurrent neural network using LSTMs
Uses Pytorch as ML framework
## Description
Under refactoring, README will be updated when refactoring completed.

## Running
When charLSTM.py is ran, it will begin training using the CSV dataset set by 'trainingData' variable (use your own nerd)
After training (or interruption of it) it will launch a terminal based chat interface for you to interact with the LSTM
